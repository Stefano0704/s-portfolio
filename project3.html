Gemtopia website

Group project | 6 months

Project Overview

Create a responsive website using Bootstrap based on a utopian reality. From paper sketches create wireframes and Mockups using Figma. Design a logo, create all the images with AI tools and find a business for your website. Focus on the UX and usability.

It was a group project with six people, each assigned a specific role. I was responsible for the designing the user interface, creating the website pages from sketches to wireframes and mockups. Collaboration and support to the coding team was constant.
